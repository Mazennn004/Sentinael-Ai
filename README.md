
## What is Sentinel AI?
Sentinel AI is an **AI-powered smart accident response system** built as a graduation thesis project at the **Faculty of Computers and Information Technology, Future University in Egypt (FUE)**, Academic Year 2025–2026.

The system automates the full lifecycle of a road accident — from real-time detection using smartphone sensors, to dashcam footage retrieval, AI-powered video analysis, and structured official report generation. No human intervention needed.
